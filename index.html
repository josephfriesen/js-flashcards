Javascript (JS)

JavaScript (often abbreviated JS) is a high-level, interpreted programming language. It is a language which is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm. (source: Wikipedia)

Operators

A construct supported by a given programming language, which behaves like a function but differs syntactically or semantically from usual functions. E.g. "+", "<", and "=".

Variables

A storage location (identified by a memory address) paired with an associated symbolic name (an identifier), which contains some known or unknown quantity of information referred to as a value.

Variable Naming Convention

A set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentation.

Functions

A specific procedure or routine performed by a program.

Methods

A procedure that is defined as part of a class and included in any object of that class, in OOP (Object-Oriented Programming). A method can only have access to data known to that object, which ensures data integrity among the sets of objects in an application. A method can be re-used in multiple objects.

Arguments

A variable passed to a function when it (the function) is called.

Parameters

A special kind of variable, used in a subroutine to refer to a piece of data provided as input to the subroutine. These pieces of data are the values of the arguments (often called actual arguments or actual parameters) with which the subroutine is going to be called/invoked.

Return

The point at which the current subroutine finishes its execution, and resume the code immediately after where it was called, known as its return address.

Chaining Methods

A common syntax for calling multiple methods to act on a single object.

Strings

A data type composed of a finite series of characters.

Booleans

A binary variable, having two possible values called "true" and "false."

Undefined

The designation of an incorrect value produced by a syntactically correct expression.

NaN

"Not a Number," i.e. a data type value respresenting an undefined or unrespresentable value.

Escape

The means by which to invoke an alternative interpretation (i.e. literal) on subsequent characters in a chracter sequence. An escape character is a particular case of metacharacters.

Alert

A "one-way" dialogue box which does not allow for input from the user.

Comments

An explanatory note embedded within code.

jQuery

A cross-platform JavaScript library designed to simplify the scripting labor of HTML.

Attributes

In computing, an attribute is a specification that defines a property of an object, element, or file. An attribute of an object usually consists of a name and a value; of an element, a type or class name; of a file, a name and extension. For clarity, attributes should more correctly be considered metadata.
